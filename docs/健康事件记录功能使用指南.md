# 健康事件记录功能使用指南

## 功能简介

健康事件记录功能让您能够记录日常的健康管理活动，包括：
- 📅 每日用药记录（预防性药物）
- 🌿 中医治疗（针灸、按摩等）
- 🏥 手术记录

这些记录与偏头痛发作记录分开管理，帮助您全面了解治疗效果。

## 快速开始

### 1. 记录您的第一个健康事件

#### 从首页开始
1. 打开应用，进入首页
2. 在"开始记录"按钮下方，找到"记录健康事件"按钮
3. 点击该按钮

#### 选择事件类型
在弹出的表单中，顶部有三个选项：
- **用药** - 记录每日服用的药物
- **中医治疗** - 记录针灸、按摩等治疗
- **手术** - 记录手术信息

### 2. 记录用药事件

适用场景：每天服用预防性药物、中成药等

#### 填写步骤
1. 选择"用药"类型
2. 设置用药日期和时间
3. 选择药物：
   - **从药箱选择**：点击"从药箱选择"，从已有药品中选择
   - **手动输入**：直接在"药物名称"框中输入
4. 填写剂量和单位（如：5mg）
5. 添加备注（可选）：如"预防性用药"、"睡前服用"
6. 点击"保存"

#### 提示
- 如果药箱中没有您要记录的药物，可以手动输入
- 剂量会自动带出药箱中设置的标准剂量
- 时间默认为当前时间，可以调整

### 3. 记录中医治疗

适用场景：针灸、推拿按摩、拔罐、刮痧、艾灸等

#### 填写步骤
1. 选择"中医治疗"类型
2. 设置治疗日期和时间
3. 选择治疗类型：
   - 针灸
   - 推拿按摩
   - 拔罐
   - 刮痧
   - 艾灸
   - 中药汤剂
   - 其他（可自定义）
4. 调整治疗时长滑块（5-120分钟）
5. 添加备注（可选）：如治疗部位、医生建议等
6. 点击"保存"

### 4. 记录手术

适用场景：偏头痛相关的手术治疗

#### 填写步骤
1. 选择"手术"类型
2. 设置手术日期和时间
3. 填写手术名称（必填）
4. 填写医院名称（可选）
5. 填写医生姓名（可选）
6. 添加备注（可选）
7. 点击"保存"

## 查看和管理记录

### 在记录列表中查看

1. 点击底部导航栏的"记录"标签
2. 默认显示所有记录（偏头痛发作 + 健康事件）混合展示

#### 识别不同类型的记录

**偏头痛发作记录**：
- 左侧显示疼痛强度数字（0-10）
- 橙色/红色色调

**健康事件记录**：
- 左侧显示事件类型图标
- 用药：蓝色药瓶图标
- 中医治疗：绿色叶子图标
- 手术：蓝色医疗箱图标

### 使用筛选器

1. 点击右上角的筛选按钮
2. 在"记录类型"部分，选择：
   - **全部记录** - 显示所有内容
   - **偏头痛发作** - 只显示发作记录
   - **健康事件** - 只显示健康事件
   - **仅用药记录** - 只显示用药事件
   - **仅中医治疗** - 只显示中医治疗
   - **仅手术记录** - 只显示手术记录

3. 可以结合其他筛选条件：
   - 时间范围（本月、近3个月等）
   - 搜索（搜索药物名称、治疗类型等）

### 查看详情

- 点击任意健康事件记录
- 查看完整的事件信息
- 对于用药事件，可以看到药物详细信息
- 对于中医治疗，可以看到治疗类型和时长
- 对于手术，可以看到医院和医生信息

### 删除记录

- 在记录列表中，向左滑动记录
- 点击"删除"按钮
- 确认删除

## 查看统计分析

### 打开数据分析

1. 点击底部导航栏的"数据"标签
2. 确保选择的是"图表"视图（顶部切换器）
3. 向下滚动查看健康事件统计

### 统计内容

#### 用药依从性卡片

显示您的用药规律性：
- **依从率**：大数字显示百分比
  - 绿色≥80%：用药很规律
  - 橙色≥60%：偶尔遗漏
  - 红色<60%：经常遗漏
- **用药天数**：实际服药的天数
- **遗漏天数**：应该服药但未记录的天数
- **环形进度图**：直观显示依从率

#### 中医治疗统计卡片

显示中医治疗的使用情况：
- **总治疗次数**：累计治疗次数
- **平均治疗时长**：每次治疗的平均时间
- **治疗类型分布**：各类治疗的频次和占比

#### 治疗效果分析卡片

显示治疗开始前后的对比：
- **开始日常用药治疗后**：
  - 治疗前30天的发作天数
  - 治疗后30天的发作天数
  - 发作减少情况（绿色提示表示有改善）
  
- **开始中医治疗后**：
  - 治疗前30天的发作天数
  - 治疗后30天的发作天数
  - 发作减少情况

这个分析帮助您了解：
- 用药是否有效
- 中医治疗是否有帮助
- 是否应该调整治疗方案

### 调整统计时间范围

- 点击右上角的时间范围选择器
- 选择：本月、近3个月、近6个月、近1年、自定义
- 统计数据会自动更新

## 导出数据

### 导出完整健康数据

1. 点击"个人"标签
2. 找到"数据导出"选项
3. 选择时间范围
4. 选择 CSV 格式
5. 查看数据预览（会显示健康事件数量）
6. 点击"导出数据"
7. 选择分享方式（AirDrop、邮件、保存到文件等）

### 导出内容包括

**统计部分**：
- 偏头痛发作统计（发作次数、平均强度等）
- 健康事件统计（用药依从性、中医治疗统计）

**详细记录部分**：
- 所有偏头痛发作的详细信息
- 所有健康事件的详细信息（用药、治疗、手术）

## 使用场景示例

### 场景 1：每日预防性用药

**目标**：记录每天服用的盐酸氟桂利嗪

**操作**：
1. 每天睡前，打开应用
2. 点击"记录健康事件"
3. 选择"用药"
4. 从药箱选择"盐酸氟桂利嗪"
5. 确认剂量（5mg）
6. 点击保存

**效果**：
- 在记录列表中看到每天的用药记录
- 在统计中看到用药依从率
- 了解自己是否按时服药

### 场景 2：每周针灸治疗

**目标**：记录每周一次的针灸治疗

**操作**：
1. 针灸治疗后，打开应用
2. 点击"记录健康事件"
3. 选择"中医治疗"
4. 选择"针灸"
5. 调整治疗时长（如30分钟）
6. 在备注中记录：百会、风池、合谷穴
7. 点击保存

**效果**：
- 记录每次针灸的时间
- 在统计中看到针灸频次
- 在治疗效果分析中，看到针灸前后的发作对比

### 场景 3：手术治疗记录

**目标**：记录偏头痛神经阻滞手术

**操作**：
1. 手术后，打开应用
2. 点击"记录健康事件"
3. 选择"手术"
4. 填写手术名称
5. 填写医院和医生
6. 在备注中记录手术效果、注意事项
7. 点击保存

**效果**：
- 完整保存手术信息
- 在时间轴上标记重要的医疗事件
- 导出数据时包含完整的医疗历史

## 最佳实践

### 1. 养成记录习惯
- 每天固定时间记录用药（如睡前、早晨）
- 治疗后立即记录，避免遗忘
- 使用备注记录额外信息

### 2. 利用统计功能
- 定期查看用药依从性，保持规律用药
- 观察治疗效果分析，了解治疗是否有效
- 如果发作未减少，及时咨询医生调整方案

### 3. 数据导出
- 就医前导出数据，提供给医生参考
- 定期备份数据，避免丢失
- 导出的 CSV 文件可以在 Excel 中进一步分析

### 4. 与发作记录结合
- 发作时记录用药，使用偏头痛记录功能
- 日常预防用药，使用健康事件记录
- 两种记录结合，获得完整的治疗画像

## 常见问题

### Q: 健康事件记录和偏头痛记录有什么区别？

**A**: 
- **偏头痛记录**：记录发作时的情况（疼痛强度、症状、诱因、发作期用药等）
- **健康事件记录**：记录日常的治疗和健康管理（预防性用药、中医调理、手术等）

### Q: 我应该在哪里记录用药？

**A**:
- **发作时的用药**（如布洛芬缓解疼痛）→ 在偏头痛记录中添加
- **日常预防性用药**（如每天的盐酸氟桂利嗪）→ 在健康事件中记录

### Q: 如何知道我的治疗是否有效？

**A**: 
在"数据"标签页中，查看"治疗效果分析"卡片：
- 对比治疗前后的发作天数
- 对比治疗前后的疼痛强度
- 绿色提示表示有改善

### Q: 可以批量记录吗？

**A**: 
目前需要逐条记录。如果您需要补记过去几天的用药：
- 逐条添加，调整日期到对应的时间
- 建议养成每日记录的习惯，避免遗漏

### Q: 记录的数据安全吗？

**A**: 
- 所有数据存储在您的设备本地
- 如果启用了 iCloud 同步，数据会加密同步到您的 iCloud
- 导出的数据由您自己管理和保存

## 数据分析建议

### 用药依从性分析

**依从率 ≥ 80%**：
- 说明：用药很规律，继续保持
- 建议：继续当前的用药计划

**依从率 60-80%**：
- 说明：偶尔遗漏用药
- 建议：设置闹钟提醒，养成固定时间服药的习惯

**依从率 < 60%**：
- 说明：经常遗漏用药
- 建议：与医生讨论是否需要调整用药方案

### 治疗效果分析

**发作天数减少 ≥ 5天**：
- 说明：治疗效果显著
- 建议：继续当前治疗方案

**发作天数减少 1-4天**：
- 说明：有一定改善
- 建议：继续观察，可以延长观察期

**发作天数无变化或增加**：
- 说明：当前治疗方案可能不够有效
- 建议：咨询医生，考虑调整治疗方案

### 中医治疗效果

观察中医治疗的频次和发作的关系：
- 治疗后1-2周内发作是否减少
- 哪种治疗类型对您更有效
- 治疗间隔多久时效果最好

## 与医生沟通

### 就诊时的数据展示

导出完整健康数据，向医生展示：

1. **治疗历史**：
   - 服用了哪些药物
   - 用药是否规律
   - 尝试了哪些中医治疗

2. **治疗效果**：
   - 治疗前后的发作对比
   - 各种治疗的效果评估

3. **完整画像**：
   - 发作记录 + 治疗记录
   - 帮助医生全面了解您的情况
   - 制定更精准的治疗方案

## 进阶使用

### 建立治疗日记

结合健康事件记录和偏头痛发作记录：

**每日记录**：
- 早晨：记录昨晚是否服药
- 发作时：记录偏头痛发作
- 治疗时：记录中医治疗

**每周回顾**：
- 查看本周用药依从性
- 查看本周发作情况
- 评估治疗效果

**每月总结**：
- 导出月度数据
- 分析治疗与发作的关联
- 调整下月治疗计划

### 数据备份

建议定期导出数据备份：
- 每月导出一次完整数据
- 重要治疗（如手术）后立即导出
- 保存到云盘或电脑，避免数据丢失
