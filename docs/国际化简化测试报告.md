## 性能影响

### 理论分析

**简化前**：
```swift
var localizedDisplayName: String {
    if isDefault && !labelKey.isEmpty {
        return String(localized: String.LocalizationValue("label.\(category).\(labelKey)"))
    } else {
        return displayName
    }
}
```
- 每次访问都需要判断 `isDefault`
- 默认标签需要动态查询本地化字符串（O(1) 但有开销）

**简化后**：
```swift
var displayName: String = "..."
```
- 直接读取存储属性（O(1) 无额外开销）
- 性能略有提升
