## 技术实现

### 字符串国际化方式

1. **简单文本**
```swift
Text(String(localized: "action.save"))
```

2. **带参数的文本**
```swift
String(format: String(localized: "stats.attack.count.format"), count)
```

3. **动态标题带计数**
```swift
let count = items.count
let title = count > 0 
    ? "\(String(localized: "form.symptoms")) (\(count)项)" 
    : String(localized: "form.symptoms")
```

4. **枚举本地化**
```swift
extension MedicationCategory {
    var localizedName: String {
        String(localized: "medication.category.\(self.rawValue)")
    }
}
```

## 维护指南

### 添加新文本时的最佳实践

1. **优先使用已有键**
   - 检查 Localizable.xcstrings 是否有相似的键

2. **新增键时遵循命名规范**
   - 使用 `category.subcategory.detail` 格式
   - 示例: `form.pain.intensity`

3. **同时添加三种语言翻译**
   - en: English
   - zh-Hans: 简体中文
   - zh-Hant: 繁体中文

4. **代码中使用**
   ```swift
   Text(String(localized: "your.new.key"))
   ```

5. **验证构建**
   - 确保Xcode编译通过
   - 在不同语言环境测试
